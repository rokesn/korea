    dramatically increasing click-through rate.

  IMPROVEMENT #3 — robots max-snippet / max-image-preview Meta
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Status: ✅ FIXED (HIGHEST IMPACT on SERP appearance)

  Before:
    <meta name="robots" content="index, follow">
    → Google truncates your description snippet at ~120 chars
    → Google crops your OG image in search results
    → Rich result eligibility reduced

  After:
    <meta name="robots" content="index, follow, max-snippet:-1,
         max-image-preview:large, max-video-preview:-1">
    <meta name="googlebot" content="index, follow, max-snippet:-1,
         max-image-preview:large">

    max-snippet:-1        → Google can show unlimited description length
    max-image-preview:large → Full-size image preview in search results
    max-video-preview:-1  → Full video previews (future-proof)

  Expected Impact:
    → Richer, longer snippets in SERP = higher CTR
    → Large image preview when sharing on KakaoTalk, Discord, etc.
    → Eligible for Google Discover cards (image-heavy feed)

  Status: ✅ FIXED

  Before: No geo targeting meta tags at all

  After: Added to homepage head():
    <meta name="geo.region"    content="KR">
    <meta name="geo.country"   content="KR">
    <meta name="geo.placename" content="대한민국">

    → Critical mitigation for the domain name mismatch (translatebahasa.pro)
